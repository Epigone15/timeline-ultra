# Timeline Ultra

Un outil simple pour créer des timelines interactives (verticales ou horizontales) avec export JSON/HTML.

## Fonctionnalités
- Ajout d'événements personnalisés
- Affichage vertical ou horizontal
- Export en JSON ou HTML
- Responsive (mobile-friendly)

## Comment utiliser ?
1. Ouvrir [index.html](https://epigone15.github.io/timeline-ultra/)
2. Remplir les champs (titre, date, description)
3. Cliquer sur "Ajouter"
4. Choisir le mode d'affichage (vertical/horizontal)
5. Exporter si besoin
